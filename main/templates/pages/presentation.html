Welcome to Welink

A Named Entity Disambiguation system for queries within Question Answering Systems.

WeLink

WeLink is a Named Entity Disambiguation system that allows linking the named entities contained in a user query to their corresponding DBpedia resources. WeLink exploits semantic and syntactic features to overcome the context shortness problem.
